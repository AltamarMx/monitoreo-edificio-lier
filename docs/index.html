Confort térmico



Es el estado en el que una persona se siente térmicamente satisfecha y no desea modificar su entorno. Depende de variables físicas (temperatura del aire, humedad relativa, velocidad del aire), así como del metabolismo, el tipo de ropa, y factores psicológicos y culturales.

IoT smartwatch para confort térmico

Objetivo
Presentar un smartwatch IoT para la recolección de datos de confort térmico basado en tecnologías abiertas.

Encuestas horarias
- Nivel de aislamiento de la ropa (clo) ¿ - Actividad metabólica (met)
- Ubicación - Sensación térmica y aceptación

Mediciones fisiológicas
- Temperatura de piel en muñeca - Frecuencia cardíaca

Hardware de bajo costo
- XIAO ESP32C3, GY-906, MAX30102, XIAO Round Display
- Publicación de datos en tiempo real en plataforma IoT

Smartwatch IoT open-source para datos de confort térmico

  • Fisiológicas: temperatura de piel (muñeca), frecuencia cardíaca
